# 책방소식 웹 프로토타입

동네 책방 작성자가 소식을 폼으로 등록하고, 발행 담당자가 검토해 개별·통합 HTML을 내려받는 인터랙티브 프로토타입입니다.

## 실행

Node.js 22.13 이상이 필요합니다.

```bash
npm install
npm run dev
```

로컬 주소는 `http://localhost:3000`입니다.

## 포함 기능

- 작성자용 책방·소식 입력 폼
- 여러 소식과 대표 이미지 등록
- 브라우저 기반 초안 및 접수 데이터 저장
- 발행 담당자용 접수함
- 수정 요청 및 승인 상태
- 개별 HTML 미리보기, 코드 보기, 다운로드
- 승인된 소식을 선택하는 통합본 편집
- 통합 HTML 미리보기, 복사, 다운로드
- 데스크톱과 모바일 반응형 화면

## 검증

```bash
npm test
```

## 프로토타입 제한

- 데이터는 현재 브라우저 `localStorage`에만 저장됩니다.
- 이미지도 브라우저에 임시 저장되므로 큰 파일이나 다수 사용자를 위한 방식은 아닙니다.
- 운영 버전에는 로그인, 역할 권한, 데이터베이스, 이미지 저장소와 템플릿 관리 기능이 필요합니다.
